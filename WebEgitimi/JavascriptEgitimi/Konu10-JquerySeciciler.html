Projeye Jquery Ekleme

3 farklı yöntem kullanarak projemize ekleyebiliriz.

1-CDN sitelerinden alınan kod satırımı sayfamıza ekleyerek hemen kullanabiliriz
2-Nuget paket yöneticisi üzerinden yükleyebiliriz.


Eğer Visual Studio ile uygulama geliştiriyorsak,
Solution explorer > ProjeAdı (sağ tık) -> Manage Nuget -> Browse -> Jquery ->Install

3-Jquery nin kendi sitesinden vaya herhangi bir siteden bilgisayara indireceğimiz js dosyasını projeye ekleyip sürükle bırakla sayfada çağırarak kullanabiliriz.
