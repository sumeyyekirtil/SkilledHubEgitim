Bootstrap Giriş

Bootstrap açık kütüphanesi

En son güncel sürümü sitedeki copy paste ile link href in içine title dan sonra yapıştırılır Twitter çalışanlarının yazdığı kod bloklarıdır. Bootstrap, HTML, CSS ve JavaScript kullanılarak yazılmış, açık kaynaklı ve ücretsiz bir front-end kütüphanesidir. Eski Twitter çalışanları Mark Otto ve Jacob Thornton tarafından oluşturulan bu kütüphane, geliştiricilere duyarlı web siteleri oluşturabilme imkanı sağlar. Bootstrap'in sunduğu tasarım şablonları, tipografi, formlar, butonlar ve navigasyon gibi arayüz bileşenleri oluşturmakta kullanılabilmektedir. Bu sayede geliştiriciler, estetik konularla uğraşmadan uygulama geliştirirken, tasarımcılar için de çekici temalar oluşturma konusunda güçlü bir temel sunmaktadır.

Bootstrap Projeye Yükleme

  1. Nuget veya Npm paket yöneticisi üzerinden
  2. Bootstrap sitesinden indirip manuel yükleyebiliriz
  3. Hazır cdn kütüphane sitelerinden alacağımız kodu sayfaya ekleyerek kullanabiliriz

Bootstrap ekranı 12 eşit parçaya bölerek kullanır. Exceldeki grid-ızgara mantığyla elemanları konumlandırır. Css Flex alt yapısını kullanır.

Bootstrap Kırılım Noktaları

Bootstrap uygulamaya bağlanılan ekran çözünürlüğünü algılayarak kullanıcıya farklı responsive rasarımlar göstermemizi sağlar.

Kırılım Noktası Ön ek Ölçü
x-small xs 576px ve altı
small sm 576px ve üstü
medium md 768px ve üstü
large lg 992px ve üstü
extra large xl 1200px ve üstü
extra extra large xxl 1400px ve üstü

Container - Kapsayıcı

container sınıfı kullanarak içeriği ortalayabileceğimiz lapsayıcı bir element kullanabiliriz.
container-fluid sınıfı kullanarak içeriği tüm ekranı kapsayan bir element kullanabiliriz.
